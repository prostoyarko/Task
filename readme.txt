Requirements:

Terraform v0.13.5 or higher
AWS user with the following policies:
  AmazonEC2FullAccess
  AmazonVPCFullAccess

Infrastructure deployment steps:

1. Set environment variable for AWS access key in command line via command "set AWS_ACCESS_KEY_ID=<your_key_id>"
2. Set environment variable for AWS access key in command line via command "set AWS_SECRET_ACCESS_KEY=<your_secret_access_key_id>"
